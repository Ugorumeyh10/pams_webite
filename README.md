# Pamela Odunna - Growth Marketing Portfolio

A professional portfolio website showcasing Pamela Odunna's expertise in growth marketing, digital strategy, and social media management.

## Features

- 🎨 Modern, professional design with beige color scheme
- 📊 Interactive data visualizations and analytics
- 🎠 Project carousels and slides
- 📱 Responsive design for all devices
- 🚀 Built with Next.js 14 and React
- ✨ Smooth animations with Framer Motion
- 📈 Data charts with Recharts

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Icons**: Lucide React

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Deployment on Render

1. Push your code to a Git repository (GitHub, GitLab, or Bitbucket)
2. Connect your repository to Render
3. Render will automatically detect the Next.js application
4. The site will be deployed and available at your Render URL

Alternatively, you can use the `render.yaml` file for configuration.

## Sections

- **Hero**: Introduction and main call-to-action
- **About**: Personal story and key statistics
- **Skills**: Comprehensive skill breakdown with progress bars
- **Experience**: Professional timeline with achievements
- **Projects**: Interactive carousel showcasing featured projects
- **Social Media**: Portfolio of social media handles built
- **Analytics**: Data visualizations and campaign performance
- **Contact**: Contact form and information

## Customization

Update the content in each component file to reflect your actual:
- Work experience
- Projects and achievements
- Social media handles
- Contact information
- Analytics data
